## 💡 项目概述

SchwarzRAG 是基于深度文档理解构建的智能检索系统，专注于 Schwarz Crystal 领域的发展历程研究、知识检索与专业辅助。作为垂直领域的大语言模型应用，系统提供对话交互、智能代理和信息查询等功能。

## 🔥 近期更新

- 2024-03-23 整合 Schwarz Crystal 文献与Embedding技术，完成领域知识库构建
- 2024-03-22 实现Windows平台 Ollama+DeepSeek+Ragflow 本地化部署方案
- 2024-03-21 完成 Linux 平台 DeepSeek-R1-Distill-Qwen-7B模型 FastAPI部署，集成Langchain框架与WebDemo

## 🎉 核心功能

### 🍭 精准输入输出

- 通过深度文档理解技术，从复杂格式的非结构化数据中提取关键信息

### 🌱 可信知识溯源

- 构建 Schwarz Crystal 领域专属知识体系
- 提供答案引用溯源功能，支持文献快照与原始出处追踪

### 🍔 多源数据兼容

- 支持Word、PPT、Excel、TXT、PDF等文档格式
- 兼容图片、影印件、网页及结构化数据等异构数据源

### 🛀 智能化工作流

- 支持灵活配置LLM大语言模型与向量模型
- 提供API接口与可视化交互界面

## 🎬 技术路线

### 📝 全开源低代码方案（初级阶段）

- 基础架构：Ollama + RAGFlow
- 基座模型：DeepSeek-R1-8B
- 快速原型开发与验证

### 🚀 本地化高代码方案（目标阶段）

- 服务部署：FastAPI + vLLM 推理优化
- 框架集成：LangChain 智能体编排
- Web应用：Gradio/Streamlit 交互界面
- 数据处理：RAGFlow Parser 文档解析引擎
